## 🚀 升级指南

由于小说目录默认储存在插件目录，所以升级后会不同步旧版本的小说。

同步旧版本小说步骤：

1. `F1` 运行 `z-reader:打开本地文件目录`
2. 找到 `**\.vscode-insiders\extensions` 目录下 `aooiu.z-reader-${版本号}\book`
3. 复制对应版本的文件到当前版本的小说目录里面

v1.0.1 新增设置项 `本地小说目录`, 指定小说目录用于避免升级导致的需要手动同步问题 -->
